# Text Editor

Text editor using windows form application in c#. 

## Getting Started

Clone and open with Visual Studio 2019.

### Prerequisites

Use Visual Studio to open the project.

### Installing

1) Clone project
2) Open "Build" folder 
3) Open TextEditor.exec
